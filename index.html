Zoom Project - Hand Gesture Demo

Project Preview

This project uses Python, OpenCV, and HandTrackingModule to zoom an image based on hand gestures. Since GitHub Pages cannot run Python code, this is a demo version of the project.

How It Works

- The project detects your hands using the webcam.
- Pinch gestures (thumb + index finger) control the zoom of the image.
- The image scales smoothly based on hand distance.
